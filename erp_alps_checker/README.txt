======================================
ERP ↔ Alps 출고수량 검증 프로그램
======================================

[ 사용 방법 - 처음 설치 시 ]

1. Python 설치 (없는 경우)
   https://www.python.org/downloads/
   → Python 3.10 이상 설치
   → 설치 시 "Add Python to PATH" 반드시 체크

2. build.bat 실행 (더블클릭)
   → 자동으로 패키지 설치 및 .exe 파일 생성
   → dist\ERP_Alps_검증.exe 생성됨

3. dist\ERP_Alps_검증.exe 더블클릭하여 실행

--------------------------------------
[ 이후 실행 시 ]

dist\ERP_Alps_검증.exe 더블클릭

--------------------------------------
[ 프로그램 사용법 ]

1. ERP 파일 선택  (ERP_D1 시트 포함 Excel)
2. Alps 파일 선택 (Alps_원본 시트 포함 Excel)
3. "검증 실행" 버튼 클릭
   → 결과 자동 저장 (ERP 파일과 같은 폴더)
   → 불일치 항목 팝업 알림

--------------------------------------
[ 파일 설명 ]

gui.py         - 프로그램 메인 (UI)
logic.py       - 검증 로직
requirements.txt - 필요 패키지 목록
build.bat      - .exe 빌드 스크립트
